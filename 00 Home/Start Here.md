# Start Here

This vault is a general notebook for projects, papers, ideas, and build notes.

The idea is to move slowly:

1. Keep notes lightweight.
2. Link related things instead of over-organizing.
3. Let projects earn structure as they grow.
4. Learn Obsidian one feature at a time.

## Start path

If you are new to Obsidian, open these notes in order:

1. [[Obsidian Slow Start]]
2. [[Projects]]
3. Pick one project hub and follow its links

> [!tip]
> Anything wrapped in `[[double brackets]]` is an internal Obsidian link. Click around, then open the Backlinks pane to see which notes point back to the one you are on.

## How this vault is organized

Right now the structure is intentionally simple:

- project hubs live behind [[Projects]]
- paper notes live in `20 Papers/`
- concept notes live in `30 Concepts/`
- design and build notes live in `40 Build/`
- journals live in `50 Journal/`

If another topic grows, we can give it its own hub note the same way PPLNS has one.

## Why this shape works

- the vault stays general
- each project can still have a clear home
- links stay more important than folders
- we avoid premature system design in the note-taking setup too
